## Как добавить изображения

### Вариант 1: Скачать текущие изображения автоматически

Запустите скрипт для скачивания всех 22 изображений с ImgBB:

```bash
node scripts/download-gallery-images.js
```

Скрипт автоматически:
- Скачает все изображения с текущих URL
- Сохранит их в `public/images/gallery/`
- Назовет файлы последовательно: `gallery-01.jpg`, `gallery-02.jpg`, и т.д.

### Вариант 2: Добавить изображения вручную

1. Поместите ваши изображения в папку `public/images/gallery/`
2. Назовите файлы последовательно:
   - `gallery-01.jpg`
   - `gallery-02.jpg`
   - `gallery-03.jpg`
   - и так далее...
3. Если количество изображений изменилось, обновите массивы в `components/GallerySection.tsx`

## Структура путей

В Vite файлы из папки `public` доступны по путям, начинающимся с `/`:

- **Файл на диске**: `public/images/gallery/photo.jpg`
- **Путь в коде**: `/images/gallery/photo.jpg`

## Текущая конфигурация

- **Всего изображений**: 22
- **Формат**: JPG (первые 10) и PNG (остальные 12)
- **Превью слайдера**: 8 изображений

## Рекомендации

- Используйте оптимизированные изображения (сжатые, но с хорошим качеством)
- Рекомендуемый размер: 1200-2000px по большей стороне
- Вес файла: старайтесь не превышать 500KB на изображение
- Формат: JPG для фотографий, PNG для изображений с прозрачностью

## Проверка

После добавления изображений:
1. Запустите проект: `npm run dev`
2. Откройте страницу с галереей
3. Убедитесь, что все изображения загружаются корректно
