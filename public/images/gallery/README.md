# Галерея изображений

Эта папка содержит фотографии для галереи сайта.

## Как добавить изображения

1. Поместите ваши изображения (JPG, PNG) в эту папку
2. Назовите файлы последовательно: `gallery-01.jpg`, `gallery-02.jpg`, и т.д.
3. Обновите массив `allGalleryImages` в файле `components/GallerySection.tsx`

## Текущие изображения

Сейчас в галерее используется 22 изображения. Все они должны быть размещены в этой папке.

## Рекомендации

- Используйте оптимизированные изображения (сжатые, но с хорошим качеством)
- Рекомендуемый формат: JPG для фотографий, PNG для изображений с прозрачностью
- Рекомендуемый размер: 1200-2000px по большей стороне
- Вес файла: старайтесь не превышать 500KB на изображение
